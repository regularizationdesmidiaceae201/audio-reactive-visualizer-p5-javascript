## 🎵 What this app does

audio-reactive-visualizer-p5-javascript is a browser-based visualizer that reacts to music and sound. It uses p5.js to draw moving shapes, color shifts, and loopable scenes that change with the audio.

You can load audio, start playback, pause it, and watch the visuals respond in real time. The app runs in a browser on Windows after you download the release package from the link above.

This project adapts an original Java and Processing visualizer for the browser. It keeps the audio-reactive idea and adds simple playback controls that work in a web setting.

The goal is to give you a clean way to run loopable audio visuals on Windows without needing to set up a coding environment.
